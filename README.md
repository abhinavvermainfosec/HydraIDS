# HydraIDS

A lightweight Python-based Intrusion Detection System (IDS) built from scratch.

## Features
- Live packet capture
- Port scan detection
- SYN flood detection
- ICMP flood detection
- CLI interface
- Alert logging

## 🚀 Installation

```bash
git clone https://github.com/abhinavvermainfosec/HydraIDS.git
cd HydraIDS

python3 -m venv .venv
source .venv/bin/activate

pip install -r requirements.txt

sudo .venv/bin/python main.py
```
